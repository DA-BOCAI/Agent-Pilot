# 接口调试与测试说明

## 1. Swagger 访问地址
- Swagger UI：`http://localhost:8080/swagger-ui/index.html`
- OpenAPI JSON：`http://localhost:8080/v3/api-docs`

## 2. 推荐联调顺序
1. 创建任务
2. 生成规划
3. 执行到确认闸门
4. 确认步骤
5. 继续执行直到完成
6. 查询详情和事件

## 3. PowerShell 调试示例
```powershell
$base = "http://localhost:8080/api/v1/tasks"

$task = Invoke-RestMethod -Method Post -Uri $base -ContentType "application/json" -Body '{"inputText":"请生成新品发布会汇报","requestId":"req-2001","source":"im_text","userId":"u-01"}'
$taskId = $task.taskId

Invoke-RestMethod -Method Post -Uri "$base/$taskId/plan"
Invoke-RestMethod -Method Post -Uri "$base/$taskId/execute"
Invoke-RestMethod -Method Post -Uri "$base/$taskId/confirm" -ContentType "application/json" -Body '{"stepId":"C_DOC","approved":true,"comment":"继续"}'
Invoke-RestMethod -Method Post -Uri "$base/$taskId/execute"
Invoke-RestMethod -Method Post -Uri "$base/$taskId/confirm" -ContentType "application/json" -Body '{"stepId":"F_DELIVER","approved":true}'
Invoke-RestMethod -Method Post -Uri "$base/$taskId/execute"
Invoke-RestMethod -Method Get -Uri "$base/$taskId"
Invoke-RestMethod -Method Get -Uri "$base/$taskId/events"
```
